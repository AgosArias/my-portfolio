Über Mich

Hallo, ich bin Agostina Aylén Arias Di Pinto

Ich bin eine Entwicklerin mit etwa 3 Jahren Erfahrung in der Prozessautomatisierung, Datenanalyse und Backend-Entwicklung, spezialisiert auf die Optimierung von Arbeitsabläufen und Effizienzsteigerung.

  • Webentwicklung: HTML, CSS, JavaScript, React
  • Power Platform: Power Automate, Power Apps, Power BI
  • Datenanalyse: Python, Pandas, Matplotlib, NumPy
  • Backend-Entwicklung: Node.js, Django, Flask
  • Cloud: AWS (Amazon Web Services)
  • Agile Methoden: Scrum, Kanban
  • Projektmanagement-Tools: Jira, Trello
  • Automatisierung und Scripting: Python, Selenium
  • Versionskontrolle: Git, GitHub, GitLab
